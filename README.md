## 游戏介绍
《穿越要塞》是一款基于 HTML5 Canvas 开发的经典太空射击类游戏，玩家通过鼠标操控星际飞船，在宇宙中对抗一波波敌机和强力 Boss，收集道具强化自身，挑战更高分数和生存时长。游戏融合了升级系统、道具系统、Boss 战等核心玩法，兼具趣味性和挑战性。

## 核心玩法
- **操作方式**：鼠标移动控制飞船位置，飞船自动发射子弹攻击敌机
- **难度递增**：游戏时间越长，敌人数量和强度越高，每 30 秒触发一轮 Boss 战
- **成长体系**：击败 Boss 可获得永久属性升级，收集道具获得临时强力效果
- **生存目标**：尽可能躲避伤害、击败敌人，挑战最高分数记录

## 游戏系统
### 1. 升级系统（击败 Boss 解锁）
| 升级项       | 效果                     |
|--------------|--------------------------|
| 能源核心     | 提升飞船血量上限         |
| 杀戮炮弹     | 增加单颗炮弹伤害         |
| 武装强化     | 提升同时发射的炮弹数量   |

### 2. 道具系统（随机掉落）
| 道具名称     | 效果                     |
|--------------|--------------------------|
| 备用炮弹     | 激活双发模式，发射双倍子弹 |
| 无尽激光     | 切换激光武器，伤害大幅提升 |
| 斜向炮       | 向多方向发射子弹，扩大攻击范围 |
| 超级护盾     | 生成临时无敌护盾         |
| 强化机炮     | 提升射击频率             |
| 空间黑洞     | 发射黑洞，清除范围内敌机和子弹 |
| 备用能源     | 恢复飞船血量             |

### 3. 护盾机制
- 初始 2 层护盾，被击中后破碎，冷却后自动恢复
- 护盾破碎时获得短暂无敌时间，降低暴毙风险

### 4. Boss 战系统
- 每 30 秒触发一次 Boss 战，共 4 轮，难度逐级提升
- Boss 拥有高额血量和特殊攻击模式，击败后获得升级机会

## 技术实现
- 基于 HTML5 Canvas 实现游戏画面渲染和动画效果
- 原生 JavaScript 编写游戏核心逻辑（碰撞检测、AI 行为、状态管理）
- Tailwind CSS 快速实现 UI 样式和响应式适配
- 使用 requestAnimationFrame 保证游戏帧率流畅
- 矩形碰撞算法处理子弹、敌机、飞船、道具的交互逻辑
